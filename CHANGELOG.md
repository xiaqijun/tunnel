# 更新日志

## [1.0.0] - 2026-02-05

### 新增功能
- ✅ 核心内网穿透功能
  - TCP 端口转发
  - 客户端认证机制
  - 自动重连
  - 心跳保活
  
- ✅ Web 管理界面
  - 现代化响应式设计
  - 实时客户端监控
  - 流量统计展示
  - WebSocket 实时更新
  
- ✅ 高性能优化
  - 缓冲池 (sync.Pool)
  - 协程池管理
  - 零拷贝数据传输
  - 可配置缓冲区大小
  
- ✅ API 接口
  - RESTful API
  - WebSocket 推送
  - JSON 数据格式
  
- ✅ 配置管理
  - YAML 配置文件
  - 灵活的性能参数
  - 多隧道支持

### 性能指标
- 吞吐量: 1-10 Gbps
- 并发连接: 10,000+
- 延迟: < 1ms (本地)
- 内存使用: < 200MB (1000 连接)
